Introduction

Biochain-AI is a projects that aims to create a blockchain system that allows a consortium of organizations to manage biological data in a secure way. The system allows to upload data into the system and make it available for sharing requests from all the others organizations. It also allows to store the secret information about the data into a private collection in order to keep the data secure until requested. Every action performed on the system is securely stored into the blockchain which tracks every action of every organization.

About this App

The system is compose by three main sections:

  • The Blockchain
  • The Rest Api Server
  • The Web Server

The blockchain system is a series of elements that talk to eachother. It's the core of the project. It runs the chaincodes that allows to store data and the transations securely inside the ledger.
Every organization that wants to join the consortium must create its own "server" that communicate to all the others organzations. Inside it are stored the secret biological data that the organization makes available to the others organizations in the consortium.

The Web Server allows to have a web inferface to communicate with the blockchain. It is a private interface that needs authentication to be visible.

The Rest Api Server is the system that allows to connect the blockchain with the web application. It gives a Rest interface to the web server to send all the available requests. This Golang server call the correct chaincode method into the blockchain passing the necessary information.

Technology

The system is developed using different technologies.
The blockchain system is develop using the IBM framework called Hyperledger Fabric. The custom chaincodes used are written using the Golang programming language. The Rest server is written using the Golang programming language. The Web server is written in PHP.
